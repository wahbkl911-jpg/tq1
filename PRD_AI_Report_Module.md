# AI 报告模块产品需求文档 (PRD)

## 1. 模块概述

### 1.1 产品定位
AI 报告模块是一个基于大语言模型的智能报告生成系统，用户通过自然语言描述报告需求，系统自动收集数据、分析内容并生成专业的行业研究报告、企业分析报告、产品技术报告等。

### 1.2 目标用户
- 行业分析师
- 企业研究人员
- 产品经理
- 市场研究人员
- 投资分析师

### 1.3 核心价值
- **一键生成**：输入主题即可自动生成完整报告
- **数据驱动**：接入海量行业数据库，确保报告数据准确
- **专业模板**：内置多种报告模板，符合行业标准
- **实时协作**：支持多人协作编辑和评论

## 2. 功能架构

### 2.1 模块结构
```
AI报告模块
├── 报告首页 (ReportHome)
│   ├── 快速创建入口
│   ├── 优质案例展示
│   └── 最近报告列表
├── 报告对话 (ReportChat)
│   ├── 对话式报告生成
│   ├── 实时预览面板
│   └── 报告导出
├── 报告管理 (ReportList)
│   ├── 报告列表
│   ├── 分类筛选
│   └── 批量操作
├── 报告详情 (ReportDetail)
│   ├── 报告阅读器
│   ├── 目录导航
│   └── 分享下载
└── 新建报告弹窗 (CreateReportModal)
    ├── 基础信息填写
    ├── 素材添加
    └── 模板选择
```

### 2.2 路由规划
| 路由 | 页面 | 说明 |
|------|------|------|
| `/reports` | ReportHome | 报告首页 |
| `/reports/chat` | ReportChat | 对话式生成 |
| `/reports/list` | ReportList | 报告列表 |
| `/reports/:id` | ReportDetail | 报告详情 |

## 5. 数据结构

### 5.1 报告对象
```typescript
interface Report {
  id: string;
  title: string;
  theme: string;
  status: 'generating' | 'completed' | 'failed';
  content: ReportContent;
  materials: Material[];
  createdAt: string;
  updatedAt: string;
  createdBy: string;
}

interface ReportContent {
  cover: CoverInfo;
  toc: TOCItem[];
  sections: Section[];
}

interface Material {
  id: string;
  type: 'file' | 'data';
  name: string;
  source: string;
}
```

### 5.2 对话消息
```typescript
interface ReportMessage {
  id: string;
  reportId: string;
  role: 'user' | 'assistant';
  content: string;
  type: 'text' | 'file' | 'progress' | 'chart';
  metadata?: {
    files?: FileInfo[];
    progress?: number;
    chart?: ChartData;
  };
  timestamp: string;
}
```

## 6. 技术实现要点

### 6.1 前端技术栈
- React + TypeScript
- Ant Design 组件库
- React Router 路由
- Zustand 状态管理
- SSE 实时通信

### 6.2 关键功能实现
| 功能 | 技术方案 |
|------|----------|
| 流式生成 | SSE + 打字机效果 |
| 实时预览 | 双栏布局 + iframe |
| 目录导航 | IntersectionObserver |
| 图表渲染 | ECharts / AntV |
| 文件预览 | PDF.js / Office Online |

### 6.3 性能优化
- 虚拟滚动（长报告）
- 懒加载图片
- 防抖搜索
- 缓存报告内容
